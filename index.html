  1. You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string.(Using SETs)

        const s=new Set();

  2. You are given a string (STR) of length N, you have to print the count of all alphabet.(using MAPs)
    3. let word="abcadeecfb";

        let map=new Map();

        for(let i=0;i< word.length;i++){

            if(map.has(word[i])){

                map.set(word[i],Number(map.get(word[i]))+1);

            }

            else{

                map.set(word[i],1);

            }

        }

        for(const [key,value] of map){

            console.log(key + " = " + value);

        }
